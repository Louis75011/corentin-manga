# Corentin Manga | Ken le Survivant

Sous-titre : la page qui fracasse des tronches.

Site éditorial en React et Vite consacré à Hokuto no Ken, avec une page principale centrée sur Ken le Survivant et une page secondaire dédiée à Soten no Ken, son fond historique shanghaïen et ses bricolages très inspirés, mais racontés avec un humour contrôlé.

## Contenu du site

1. Une page principale qui utilise les versions v2 des visuels de Corentin et Kenshiro.
2. Une page Soten no Ken accessible depuis la navigation, avec les visuels classiques affichés en grand.
3. Des métadonnées de partage web et réseaux sociaux basées sur la bannière Kenshiro dédiée.

## Lancer le projet

Prérequis : Node.js.

1. Installer les dépendances avec npm install.
2. Démarrer le serveur de développement avec npm run dev.
3. Générer la version de production avec npm run build.

## Stack

1. React 19
2. Vite
3. TypeScript
4. Tailwind CSS
5. Motion
